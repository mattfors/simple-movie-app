### Built With

* Spring Boot
* Mybatis
* H2 Database
* Thymeleaf


## Getting Started


### Prerequisites

Install:
* Maven


### Installation

1. Start the application
   ```sh
   mvn spring-boot:run
   ```
   ```
2. Navigate to http://localhost:8080/
